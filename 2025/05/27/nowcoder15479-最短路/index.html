解题思路

  1. 道路之间的花费和城市编号的二进制位有关系
  2. 直接建图会导致 图的边数太多,进而超时,因此需要精简边数
  3. 可以发现 a xor c = a xor b xor b xor c < (a xor b) + (b xor c)
  4. 对最短路长度的贡献取决于二者不同位的个数和位置。一旦引入中间结点,又导致位数贡献变多，不是最优。
  5. 精简道路, 所有的点都连向和自己只相差一位的点，这样经过有限次位数的变动，可以使得i变成j,且每次变动的都是与不同的位数,因此答案不会变差。
  6. 新增快速通道,建图求最短路即可
